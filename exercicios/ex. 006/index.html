Praticando

Agora, vou escrever um parágrafo, usando todas as tags aprendidas anteriormente

Com a tag strong, deixo uma palavra em negrito. Com a tag em, deixo a palavra em itálico. Posso riscar uma sentença com a tag del, e inserir ou sublinhar, com a tag ins. Também posso deixar palavras sobrescritas com a tag sup, ou subescritas, com a tag sub. A tag mark é um marca texto, de forma padrão amarela, podendo mudar sua configuração de cor pelo css

Essas letras ficam pequenas com a tag small, exemplificando letras pequenas de um contrato por exemplo
